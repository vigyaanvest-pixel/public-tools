# VigyaanVest.com Investor Tools

Open-source investor utilities from [VigyaanVest.com](https://vigyaanvest.com).

VigyaanVest.com builds professional stock market signals and investment education for serious traders, with a focus on transparent tracking and "verify before you join" research workflows.

These tools are published for the investor community as inspectable, browser-first utilities. They are intentionally lightweight, local-first, and documented so users can review what they do before loading them in developer mode.

## Tools

- [Market Squawk](market-squawk/) - Microsoft Edge / Chrome extension that reads newly detected market headlines aloud from supported public news pages.
- [Nasdaq Halt Monitor](nasdaq-halt-monitor/) - Microsoft Edge / Chrome extension that monitors Nasdaq Trader current trade halts with watchlist and reason-code alerts.
