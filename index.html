Available Tools

Market Squawk
Edge / Chrome Extension

Watches supported investor news pages and reads newly detected top headlines aloud. A browser-native alternative to paid squawk services — no subscription required.

  • TradingView News Flow, Finviz, and MarketWatch
  • Smart expansion of tickers and market shorthand
  • Configurable voice, speed, volume, and watch count
  • No account, no server, no data sent anywhere
View on GitHub Docs
Nasdaq Halt Monitor
Edge / Chrome Extension

Monitors the public Nasdaq Trader trade halts page and fires browser notifications when matching halt events are detected. No paid data feed required.

  • LUDP, LUDS, T1, T2 and additional halt codes
  • Optional symbol watchlist or broad-halt mode
  • Windows toast notifications and badge alerts
  • Adjustable poll interval, open-halt filter, local alert memory
View on GitHub Docs

Our Philosophy

🔍

Verify Before You Use

Every tool is open source. Read the code, review the manifest, and understand what it does before loading it in developer mode.

🖥️

Local-First

No accounts, no servers, no external APIs. Settings and data stay in your browser's local storage and never leave your machine.

⚡

Lightweight by Design

Each extension is built to do one thing well with minimal permissions. No background analytics, no telemetry, no ads.

🛠️

Community-Inspectable

Published under GPLv3. Fork it, audit it, improve it, or contribute a new source. Open by default.
